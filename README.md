# Technology used:

- Postman:API測試
- FLASK
- Mongodb:Database
- pymongo: 讓python程式能和mongoDB資料庫溝通，可以進行資料的新增、查詢、更新、刪除(CRUD)
- bson格式: 主要被用作MongoDB資料庫中的資料儲存和網路傳輸格式
